<h1 align="center">Loja de Roupas Masculinas</h1>

<p align="center">
  Projeto desenvolvido para a disciplina de <strong>Programação Web Responsiva</strong> da <strong>UNIP</strong>.
</p>

<p align="center">
  O objetivo deste projeto foi desenvolver um site responsivo para uma loja de roupas masculinas utilizando apenas
  <strong>HTML5</strong> e <strong>CSS3</strong>, aplicando conceitos de estruturação semântica, estilização e adaptação para diferentes tamanhos de tela.
</p>

<hr>

<h2> Tecnologias Utilizadas</h2>

<ul>
  <li>HTML5</li>
  <li>CSS3</li>
</ul>

<h2> Objetivos do Projeto</h2>

<ul>
  <li>Desenvolver uma estrutura semântica utilizando HTML5.</li>
  <li>Criar uma interface moderna e organizada com CSS3.</li>
  <li>Aplicar técnicas de responsividade para diferentes dispositivos.</li>
  <li>Praticar conceitos de desenvolvimento web front-end.</li>
  <li>Organizar o código de forma limpa e de fácil manutenção.</li>
</ul>
